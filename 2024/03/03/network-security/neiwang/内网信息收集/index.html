进入内网后要对当前网络环境进行判断

判断出当前机器所处网络环境的拓扑，以及当前电脑在内网是什么角色当前机器的权限

# 本机信息收集

# 查询网络配置信息

ipconfig /all

# 查看用户详细信息

whoami /all

# 查看系统版本补丁等

systeminfo 可查看系统信息打了那些补丁为后续提权查找漏洞准备

# 查看系统启动的服务

net start

# 查看系统进程

# 查看主机计划任务列表

# 查看本机用户域用户

net user net user /domain

# 查询域数量

net view /domain

# 查询域组

net group /domain

# 查询域中的 “Domain Admins” 组的成员

net group "domain admins" /domain

# 查询域时间主域名

net time /domain

主域名为 ad01.yunyi.io

# 查 ARP 缓存

arp -a

# 查环境变量

set

# 查看安装软件的版本信息

powershell "Get-WmiObject -class Win32_Product |Select-Object -Property name,version"

# 域信任列表信息

nltest /domain_trusts

# 防火墙配置信息

netsh firewall show config

# 网络详细信息

ipconfig /all 可看到有域 yunyi.io

# 查看网络端口信息

netstat -ano

# 查杀软查提权

tasklist 查看当前计算机上所有进程的列表

systeminfo

查完补丁信息可以去 https://i.hacking8.com/tiquan/ 查询

# icmp 协议探测

win

for /L %I in (1,1,254) Do @ping -w 1 -n 1 10.10.10.%I | findstr "TTL"

linux

for k in $( seq 1 255);do ping -c 1 192.168.7.$k|grep "ttl"|awk -F "[ :]+" '{print $4}'; done
