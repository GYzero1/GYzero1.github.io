String（字符串）：用于表示文本数据的数据类型，使用单引号或双引号括起来。

Integer（整数）：用于表示整数值，可以是正数、负数或零。

Float（浮点数）：用于表示带有小数部分的数值。

Boolean（布尔值）：用于表示 true 或 false 两种状态。

Array（数组）：用于存储多个值的集合，可以是索引数组或关联数组。

Object（对象）：用于存储数据和操作的实例。

NULL（空值）：用于表示变量没有值。

Resource（资源）：特殊数据类型，表示外部资源（如数据库连接）的引用。

# 字符串

字符串通过双引号单引号包裹

使用双引号 "" 括起来的文本。双引号字符串允许解析变量和转义字符。

字符串还支持转义字符，例如：

  • \n : 换行符
  • \t : 制表符
  • \\ : 反斜杠
  • \" : 双引号
  • \' : 单引号

# 字符串处理

在 PHP 中，有许多内置函数用于处理字符串。这些函数可以用于查找、截取、替换、转换大小写等操作。

  1. strlen()：获取字符串长度。
  2. strpos()：查找子串在字符串中第一次出现的位置。
  3. substr()：从字符串中提取子串。
  4. str_replace()：替换字符串中的部分内容。
  5. strtoupper() / strtolower()：将字符串转换为大写 / 小写。
  6. trim()：去除字符串两端的空格或特定字符。
  7. explode() / implode()：将字符串拆分为数组 / 将数组元素连接成字符串。
  8. printf() / sprintf()：格式化字符串输出。
  9. str_split()：将字符串拆分为字符数组
